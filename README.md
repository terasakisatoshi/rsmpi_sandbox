# rsmpi_sandbox

このリポジトリでは rsmpi を MPItranmpoline をバックエンドとしてビルドする環境とビルド済みのバイナリを実行する実行環境を提供します．

## Installation

### Docker

On macOS, refer the following instruction:

- [Install Docker Desktop on Mac](https://docs.docker.com/desktop/setup/install/mac-install/)

### `devcontainer` command

On macOS, you can install `devcontainer` command using the following command:

```zsh
% brew install devcontainer
```
